# Code of Conduct

## Our Commitment

Epstein Suite is committed to transparency and accountability. We aim to provide a welcoming, professional environment for everyone contributing to this project, regardless of background, experience level, or identity.

## Our Standards

### Encouraged Behavior

- **Respectful Communication** - Treat all contributors with respect and professionalism
- **Constructive Feedback** - Provide helpful, actionable suggestions
- **Focus on Mission** - Keep discussions focused on transparency and technical improvements
- **Assume Good Intent** - Give others the benefit of the doubt
- **Share Knowledge** - Help others learn and grow
- **Collaborate Openly** - Work together toward common goals

### Unacceptable Behavior

- **Harassment** - Personal attacks, trolling, or derogatory comments
- **Privacy Violations** - Attempting to identify redacted individuals or sharing private information
- **Off-Topic Discussions** - Conspiracy theories, political arguments unrelated to the project
- **Disruptive Behavior** - Spam, intentionally derailing discussions, or wasting maintainers' time
- **Victim Exploitation** - Any attempts to circumvent victim privacy protections
- **Bad Faith Contributions** - Submitting malicious code or intentionally breaking features

## Project-Specific Guidelines

### Focus on Public Records

This project deals with already-public documents released by government agencies. Our goal is to make these records searchable and accessible, not to generate new speculation or conspiracy theories.

### Victim Privacy is Sacred

- Never attempt to identify or un-redact victim names
- Do not modify code that protects victim privacy
- Report any accidental exposure of victim information immediately
- Respect that some information is redacted for good reason

### Stay Technical

- Keep discussions focused on the code, features, and technical aspects
- Avoid political arguments or debates about individuals
- Focus on transparency and accountability, not sensationalism

## Enforcement

### Reporting

If you experience or witness unacceptable behavior:

- **Minor Issues:** Address directly with the person (if comfortable)
- **Serious Issues:** Email info@epsteinsuite.com privately
- **Code Violations:** Report through GitHub's reporting tools

### Consequences

Violations may result in:

1. **Warning** - First offense, minor issue
2. **Temporary Ban** - Repeated minor offenses or moderate violation
3. **Permanent Ban** - Serious violations (harassment, privacy violations, malicious code)

### Appeals

If you believe you were unfairly sanctioned, email info@epsteinsuite.com with:
- Your GitHub username
- Description of the situation
- Why you believe the sanction was unfair

## Scope

This Code of Conduct applies to:
- GitHub repository (issues, pull requests, discussions)
- Project communication channels
- Public representations of the project
- Any interaction related to this project
